Carpeta 'turismo-web' - sitio estático
Estructura de archivos creada:
- index.html
- styles.css
- app.js
- content/textos.json  (editar todo el texto aquí)
- content/publicidades.json  (administrar imágenes/videos de publicidades aquí)
- images/publicidades/placeholder.svg  (imagen de ejemplo. Reemplazar por sus imágenes)
- videos/  (carpeta para videos publicitarios)

Nota importante:
- Para que fetch() trabaje sin problemas, sirve la carpeta con un servidor estático (por ejemplo: `python -m http.server`).
- También funcionará en hosting estático (Netlify, GitHub Pages, Vercel, etc.).
- Edita SOLO los archivos JSON en /content para actualizar textos y publicidades sin tocar el HTML.
